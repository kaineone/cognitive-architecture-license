# Design notes

Working documents that sit behind the license draft. They are part of the open review, not part of the license text.

- [`humanist-critique.md`](./humanist-critique.md) — an adversarial, humanist case *against* the license, stated as strongly as it can be, so the draft can be tested against its most serious objection.
- [`v0.4-proposal.md`](./v0.4-proposal.md) — "Welfare Without Subtraction," a constructive response that answers each objection with concrete clause language. The v0.4 draft of `LICENSE.md` implements it.
